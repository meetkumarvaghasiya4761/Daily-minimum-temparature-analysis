# 📊 Time Series Forecasting: Temperature Analysis

## 📌 Overview
This project focuses on analyzing daily temperature data using time series forecasting techniques. The goal is to identify patterns, understand seasonal trends, and predict future temperature values using machine learning models.

---

## 🎯 Objectives
- Analyze historical temperature data  
- Identify trends and seasonal patterns  
- Perform feature engineering  
- Build and compare predictive models  
- Forecast future temperature values  

---

## 🛠️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- Statsmodels  

---

## 📂 Dataset
- Daily minimum temperature dataset (1981–1990)  
- Provided by university  

---

## 🔍 Workflow
1. Data Cleaning and Preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Feature Engineering (lag features, time-based features)  
4. Model Development (Linear Regression, Random Forest)  
5. Model Evaluation and Prediction  

---

## 📊 Model Performance

| Model              | MSE  | MAE  | R²   |
|-------------------|------|------|------|
| Linear Regression | 6.08 | 1.93 | 0.62 |
| Random Forest     | 7.77 | 2.20 | 0.52 |

**Best Model:** Linear Regression  

---

## 📈 Key Insights
- Strong correlation between current and previous day temperature  
- Clear seasonal patterns observed  
- Lag features significantly improve prediction accuracy  
- Temperature trends are stable and predictable
